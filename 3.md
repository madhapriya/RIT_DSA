# Maximum Aqua Curtains in a Box

A furnishing company is manufacturing a new collection of curtains. The curtains are of two colors: **aqua (a)** and **black (b)**.

The curtain colors are represented as a string `str` consisting of `a` and `b` characters of length **N**.

The curtains are packed into boxes, where each box contains **L curtains** (substring of length L).

The box with the **maximum number of aqua (`a`) curtains** is labeled.

Your task is to **find the number of aqua (`a`) curtains in the labeled box**.

---

## Note

If **L is not a multiple of N**, the remaining curtains should also be considered as another set.

In simple terms, after dividing the string into groups of size **L**, any leftover characters form another group.

---

## Example 1

### Input
bbbaaababa

3

### Explanation

Dividing the string into sets of length **3**:

Set 1: `{b,b,b}`  
Set 2: `{a,a,a}`  
Set 3: `{b,a,b}`  
Set 4: `{a}` (remaining characters)

Among all sets, **Set 2** contains the maximum number of `a`.

Number of `a` in Set 2 = **3**

### Output
3


---

## Example 2

### Input

abbbaabbb

5


### Explanation

Dividing the string into sets of length **5**:

Set 1: `{a,b,b,b,b}`  
Set 2: `{a,a,b,b,b}`  

Set 2 contains more `a`.

Number of `a` in Set 2 = **2**

### Output
2


---

## Constraints

- `1 ≤ L ≤ 10`
- `1 ≤ N ≤ 50`

---

## Input Format

1. First line: String `str` containing only characters `a` and `b`.
2. Second line: Integer `L` representing the number of curtains in each box.

---

## Output Format

Print a **single integer** representing the **maximum number of `a` characters** in any box.
